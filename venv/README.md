# Enterprise Backend Project

## Overview
This is a FastAPI backend project with CRUD APIs and SQLite database integration.

## Features
- FastAPI backend
- CRUD APIs
- Swagger documentation
- SQLite database
- SQLAlchemy ORM

## Technologies
- Python
- FastAPI
- SQLAlchemy
- SQLite

## Run Project

Activate virtual environment:

source venv/bin/activate

Run server:

uvicorn main:app --reload

http://127.0.0.1:8000/docs
